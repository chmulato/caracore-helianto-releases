Helianto Condominium

Administração e Soberania de Dados na Reconstrução do SaaS Condominial.

Resumo técnico

Java 25, Spring Boot 4.0.6, PostgreSQL Multi-tenant. Automação de juros e multas por rateio pro rata die, BolePix integrado e segurança de logs com PrivacyGuard (LGPD Shield).

Esta é a wiki de releases da loja (chmulato.github.io/caracore-helianto-releases). O manual de implantação completo está disponível em: manual nesta loja.

Arquitetura de Isolamento de Dados

O back-end do Helianto Condominium assegura isolamento lógico absoluto no banco PostgreSQL através de filtros automáticos de tenant baseados no cabeçalho HTTP do token JWT. O isolamento atende a requisitos regulamentares da LGPD, prevenindo vazamentos de relatórios contábeis e informações pessoais.

Regras de Negócio e Cálculos Contábeis

Diferente de sistemas legados que utilizam juros simples ou planilhas suscetíveis a erros humanos, o motor financeiro do Helianto realiza os cálculos de multa de 2% e juros de 1% ao mês pro rata die via código compilado seguro, gerenciando contas separadas para fundos ordinários, extraordinários e de reserva.
