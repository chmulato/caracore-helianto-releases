Helianto Condominium

Administração e Soberania de Dados na Reconstrução do SaaS Condominial

O Produto

O Helianto Condominium é uma plataforma SaaS projetada a partir de uma folha em branco para resolver gargalos do mercado imobiliário e de gestão condominial. Com foco em estabilidade, segurança e integridade financeira sob a ótica FinOps, ele substitui caixas-pretas por um sistema auditável, transparente e escalável.

Cálculo de Rateio Flexível

Suporta o cálculo da taxa condominial tanto por fração ideal quanto por divisão igualitária de despesas.

Régua de Cobrança Precisa

Executa cálculos automáticos de multa fixa de 2% e juros de 1% ao mês pro rata die (Código Civil brasileiro).

Separação de Fundos

Diferenciação contábil clara entre despesas ordinárias e extraordinárias (fundos de obras e fundo de reserva).

BolePix Nativo

Pagamento instantâneo via QR Code Pix ou código de barras tradicional para reduzir taxas e acelerar o caixa.

Isolamento Multi-Tenant

Dados de cada condomínio estão matematicamente isolados no PostgreSQL com contexto de tenant por requisição.

LogSanitizer (LGPD)

Mecanismo automático que higieniza telefones, CPFs/CNPJs e hashes de IP nos logs em tempo real sem perda de performance.

Especificações Técnicas

Plataforma: Nuvem / VPS Linux
Tecnologia: Java 25 (LTS) + Spring Boot 4.0.6
Banco de Dados: PostgreSQL Multi-Tenant
Arquitetura: API-first / Web responsiva
Deploy: Docker Compose + Caddy / Nginx
Testes: Cobertura de 81% (JaCoCo)
